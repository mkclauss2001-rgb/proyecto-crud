# CRUD Básico

Este proyecto implementa un **CRUD (Create, Read, Update, Delete)** sencillo para gestionar registros en una base de datos.  
Es ideal como ejemplo educativo o punto de partida para proyectos más complejos.

---

## Características
- Crear nuevos registros
- Listar registros existentes
- Editar registros
- Eliminar registros
- Conexión a base de datos relacional (MySQL)
